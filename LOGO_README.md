# Logo für Elastic Pulse Studio

## Dateiname
**`logo.png`**

## Pfad
`/public/logo.png`

## Verwendung
Das Logo wird beim App-Start auf der Leinwand (Canvas) angezeigt und verschwindet automatisch, sobald die Timeline startet.

## Funktionen
- **Automatisches Verschwinden**: Das Logo verschwindet, sobald die Session/Timeline startet
- **Hold-Button**: Mit dem "Hold"-Button kann das Logo gehalten werden, auch wenn die Timeline läuft
- **Fade-Animation**: Sanftes Ein- und Ausblenden mit 1 Sekunde Übergang

## Empfohlene Spezifikationen
- **Format**: PNG (mit Transparenz für besseres Aussehen)
- **Auflösung**: Mindestens 800x800px (für scharfe Darstellung)
- **Hintergrund**: Transparent (empfohlen)
- **Dateigröße**: Optimiert für Web (< 200KB empfohlen)

## Positionierung
Das Logo wird zentriert auf der Leinwand angezeigt und nimmt maximal 60% der Breite/Höhe ein.

## Fallback
Falls das Logo nicht gefunden wird, wird automatisch ein Fallback mit dem Pulse-Icon und "Elastic Pulse" Text angezeigt.
