# Public Assets

Dieser Ordner enthält statische Assets, die direkt über die URL verfügbar sind.

## App Screenshot

Legen Sie hier Ihr Screenshot-Bild der App ab:

**Dateiname:** `app-screenshot.png`

**Pfad:** `/public/app-screenshot.png`

**Verwendung:** Das Bild wird automatisch in der Landing Page angezeigt.

### Empfohlene Spezifikationen:
- Format: PNG oder JPG
- Auflösung: Mindestens 1920x1080px (Full HD)
- Dateigröße: Optimiert für Web (< 500KB empfohlen)

### So erstellen Sie den Screenshot:
1. Öffnen Sie die App im Browser (http://localhost:3000)
2. Klicken Sie auf "Launch Studio"
3. Machen Sie einen Screenshot der gesamten App-Oberfläche
4. Speichern Sie das Bild als `app-screenshot.png` in diesem Ordner

Das Bild wird dann automatisch in der "See It In Action" Sektion der Landing Page angezeigt.
